# Edition #002

## Stop Writing Promotional Emails. Build an AI Agent That Does It For You.

This project demonstrates how to build an AI-powered promotional email workflow that reads customer purchase data, reviews category-based discounts, generates personalized email content, and sends promotional emails automatically.

## Included
- Workflow export package
- Sample datasets
- Workflow screenshots
- AI prompt examples
- Step-by-step implementation

## Workflow Overview

Monthly Trigger  
→ Read Customer Data  
→ Review Discount Products  
→ Match Product Categories  
→ Build Discount Summary  
→ Generate AI Email  
→ Send Personalized Promotion Email
